Maze Game (React.JS/Javascript)

This Single-Page Application was written in React.JS as a way to interect with hundreds/thousands of online concurrent users. It displays a few counting/ticking components all at once. This is currently (2025) still a work in progress. It is basically complete, except for some minor tweaks before going into a Production environment. So, more to follow.

TradingCardz (Java)

Using OpenCV for image capture and Gecko/Selenium for dynamic web parsing, this application stores my sports cards in an Oracle DB with respective info and image, to be sold online. It goes out and grabs pricing, and suggests a lower price for me to tag items with before submitting them for sale online. It keeps track of the item and notifies me if there's something needing my attention.

Hashboard (java)

A hacking dashboard/toolkit that breaks down the 5 phases involved in a typical hacking process. A colorful graphical display will only show each phase's description and any associated known hacking tools. No hacking is actually performed, as it is only static/non-linked info.
Information mostly from https://www.geeksforgeeks.org/5-phases-hacking/
and
http://blog.tofte-it.dk/ethical-hacking-tools-for-the-5-phases-of-hacking/).
